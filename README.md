# @topostitch/core

The core runtime powering TopoStitch.

This package provides the platform-agnostic foundation for building contextual 3D experiences.

## Goals

- Context-first architecture
- Platform agnostic
- Framework agnostic
- Extensible
- TypeScript first

## Supports

- Web
- Mobile
- XR
- Digital Twins
- Museums
- Commerce
